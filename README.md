* vue.js
* sass
* webpack
* fontawsome

### (demo)[https://www.baidu.com]

### 本地测试
需要node与npm支持

克隆远程库
``` shell
git clone https://github.com/ycwalker/plateform
```
移动到plateform目录下 

``` bash
cd plateform
```

安装依赖
``` bash
npm install
```
```bash
npm run generate //webpack打包生成
mpm run start //开启webpack server服务器
```
结束
打开localhost:8080，即可进入登陆页面
默认内置账户 ycwalker , 密码 111
